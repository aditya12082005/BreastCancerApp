# 🧠 Breast Cancer Prediction App

A simple machine learning web app built using **Streamlit** that predicts whether a tumor is malignant or benign based 
on user input.

## 📌 Features

- 🔍 Input form for tumor features (e.g., radius, texture, area, etc.)
- 🤖 Uses a pre-trained scikit-learn model (`breastcancermodel.pkl`)
- 📊 Displays prediction result clearly
- ⚡ Fast and easy to use
